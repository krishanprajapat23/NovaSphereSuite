# NovaSphere Suite – Frontend

A lightweight landing page.
This repo includes the main layout, component sections, and UI interactions used across the site.

## Tech Stack

* **HTML5** for structure
* **CSS3 (SCSS) / Bootstrap 5** for styling and layout
* **Slick Carousel** for sliders
* **WOW.js + Animate.css** for scroll animations
* **jQuery** for small UI interactions
* **Vanilla JS** for custom behavior

## Structure

* `assets/css/` — global styles
* `assets/js/` — custom scripts
* `assets/img/` — images and icons
* `index.html` — main entry file

## Features

* Responsive design across all the devices
* Tab based content sections
* Sliders section for good visuals
* FAQ accordion
* Footer with social and store links
